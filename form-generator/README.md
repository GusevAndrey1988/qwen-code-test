# Генератор веб-форм на React 19 + Atomic Design

Интерактивное приложение для создания и настройки веб-форм с использованием архитектуры Atomic Design.

## 🚀 Возможности

### Конструктор форм
- **11 типов полей**: текст, email, пароль, число, телефон, textarea, select, checkbox, radio, дата, файл
- **Drag-and-drop управление**: перемещение полей вверх/вниз
- **Настройка параметров**: редактирование label, name, placeholder, required и специфичных настроек
- **Живой предпросмотр**: мгновенное отображение формы в режиме просмотра
- **Генерация HTML**: получение готового кода формы со стилями

### Настройка элементов
Для каждого типа поля доступны свои параметры:

#### Общие настройки (для всех полей)
- Название поля (label)
- Имя поля (name)
- Плейсхолдер
- Обязательное поле (required)

#### Специфичные настройки
- **Textarea**: количество строк
- **Select/Radio**: список опций (каждая с новой строки)
- **Number**: min, max, step
- **Text/Email/Password/Tel**: максимальная длина

## 🏗️ Архитектура Atomic Design

### Атомы (Atoms)
Базовые компоненты:
- `Button` - кнопки с вариантами styling
- `Input` - текстовые поля
- `Select` - выпадающие списки
- `Textarea` - многострочные поля
- `Checkbox` - чекбоксы

### Молекулы (Molecules)
Компоненты среднего уровня:
- `FormField` - поле формы с лейблом и контролами
- `FieldControls` - панель добавления полей
- `FieldConfigPanel` - панель настройки параметров поля

### Организмы (Organisms)
Сложные компоненты:
- `FormBuilder` - основной конструктор форм

## 🛠️ Технологии

- **React 19** - последняя версия React
- **Vite** - быстрый сборщик
- **CSS Modules** - изолированные стили
- **Atomic Design** - методология компонентного дизайна

## 📦 Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен сборки
npm run preview
```

## 💡 Использование

1. **Добавление полей**: Выберите тип поля в левой панели
2. **Настройка**: Нажмите ⚙️ на поле для редактирования параметров
3. **Перемещение**: Используйте ↑↓ для изменения порядка
4. **Предпросмотр**: Переключитесь в режим "👁️ Предпросмотр" для тестирования
5. **Экспорт**: Скопируйте HTML или скачайте готовый файл

## 🎨 Стилизация

Формы получают современный градиентный дизайн автоматически. При экспорте генерируется полный HTML с встроенными CSS стилями.
