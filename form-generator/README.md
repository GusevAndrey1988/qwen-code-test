# Генератор Веб Форм

Интерактивное приложение для создания HTML форм с визуальным предпросмотром.

## Возможности

- 📝 **11 типов полей**: текст, email, пароль, число, телефон, textarea, select, checkbox, radio, дата, файл
- 👁️ **Живой предпросмотр** формы в реальном времени
- 📋 **Генерация кода**: HTML и CSS код для вашей формы
- ⬆️⬇️ **Управление порядком**: перемещение полей вверх/вниз
- 🗑️ **Удаление полей**: удаление отдельных полей или очистка всей формы
- 💾 **Скачивание**: возможность скачать готовую форму как HTML файл
- 📋 **Копирование**: копирование HTML кода в буфер обмена
- ✅ **Валидация**: поддержка обязательных полей (required)
- 🎨 **Современный дизайн**: красивый градиентный интерфейс

## Как использовать

1. Откройте `index.html` в любом браузере
2. Выберите тип поля из выпадающего списка
3. Заполните:
   - Название поля (name атрибут)
   - Метку (label)
   - Плейсхолдер (опционально)
   - Отметьте "Обязательное поле" если нужно
   - Для select/checkbox/radio укажите опции через запятую
4. Нажмите "➕ Добавить поле"
5. Повторите для всех необходимых полей
6. Просмотрите результат во вкладке "Предпросмотр"
7. Скопируйте HTML/CSS код или скачайте готовый файл

## Структура проекта

```
form-generator/
└── index.html    # Единственный файл приложения (HTML + CSS + JS)
```

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, градиенты)
- Vanilla JavaScript (без зависимостей)

## Запуск

Просто откройте файл `index.html` в браузере. Сервер не требуется!

```bash
# Или используйте локальный сервер
python -m http.server 8000
# Затем откройте http://localhost:8000/form-generator/index.html
```
