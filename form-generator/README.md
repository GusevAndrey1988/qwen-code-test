# Генератор веб-форм на React 19 + Atomic Design

Современное приложение для создания и генерации HTML форм с использованием архитектуры Atomic Design.

## 🚀 Технологии

- **React 19** - последняя версия библиотеки
- **Vite** - быстрый сборщик проектов
- **Atomic Design** - методология компонентного дизайна
  - Atoms (Атомы) - базовые компоненты (Button, Input, Select, Textarea, Checkbox)
  - Molecules (Молекулы) - комбинации атомов (FormField, FieldControls)
  - Organisms (Организмы) - сложные компоненты (FormBuilder)

## 🛠️ Установка и запуск

```bash
cd form-generator
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## ✨ Возможности

- **11 типов полей**: текст, email, пароль, число, телефон, textarea, select, checkbox, radio, дата, файл
- **Визуальный конструктор** с живым предпросмотром
- **Управление полями**: добавление, удаление, перемещение вверх/вниз
- **Генерация HTML кода** с встроенными стилями
- **Скачивание формы** как готовый HTML файл
- **Копирование кода** в буфер обмена
- **Поддержка обязательных полей**
- **Современный градиентный дизайн**

## 📦 Сборка проекта

```bash
npm run build
```

Собранный проект будет в папке `dist/`

## 🎨 Архитектура Atomic Design

### Атомы (Atoms)
Базовые строительные блоки интерфейса:
- `Button` - кнопки различных стилей
- `Input` - текстовые поля ввода
- `Select` - выпадающие списки
- `Textarea` - многострочные текстовые поля
- `Checkbox` - чекбоксы

### Молекулы (Molecules)
Комбинации атомов, выполняющие конкретные задачи:
- `FormField` - поле формы с меткой и элементами управления
- `FieldControls` - панель добавления новых полей

### Организмы (Organisms)
Сложные компоненты, состоящие из молекул и атомов:
- `FormBuilder` - основной конструктор форм
